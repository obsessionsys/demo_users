# Create demo users with filename

ТЗ
```
Имеем папку, в которой лежат файлы, имена файлов - это имена пользователей. Нужно: написать ansible плейбук, который по имени каждого файла в этой папке, будет создавать пользователя на сервере и давать пользователю судо права на команду /opt/script.sh и запретит вход по ssh(например для запуска команды через cron) , В конце плейбук отправит письмо со списком всех добавленных пользователей.
```
